Banter: An Online Console Messaging Application

Key Features: 1. Word censorship

Technical Features:

1.  Local caching - Frequently used data throughout a session are stored in a self-contained database - Data caching starts when a user is logged in - Cleared when the session ends (a user logs out)

    2. Authentication System

       - Both username and password are managed from the database for security

    3. Interactivity
       - Mouse
         - Terminal supports mouse activities (e.g.: changing text fields can be done via mouse)
       - Keyboard
         - Supports keyboard keys for context switching (e.g.: changing text fields can be done via mouse, pressing button can be done using alt+[hotkey])
